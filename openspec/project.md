# Project Context

## Purpose
Project website for **Archi** — an open-source AI agent for CMS computing operations at CERN, built at MIT. Archi uses retrieval-augmented generation to answer operator questions with cited sources from internal documentation, JIRA tickets, and operational runbooks. Originally developed at MIT, it is deployed across CMS Computing Operations, SubMIT, Data Quality Monitoring, and Fermilab heavy-ion analysis.

This repository hosts the public-facing project website, deployed via GitHub Pages.

## Tech Stack
- Jekyll 4.3+ (direct gem, not github-pages gem — requires Ruby 3.3+)
- Markdown content with YAML front matter
- HTML/CSS with Liquid templates
- YAML data files for structured content (`_data/team.yml`, `_data/talks.yml`, `_data/publications.yml`)
- GitHub Actions for deployment (manual trigger via `workflow_dispatch`)

## Project Conventions

### Code Style
- Markdown for all blog posts
- HTML with Liquid for page templates
- Single CSS file at `assets/css/main.css` with CSS custom properties
- System sans-serif font stack, MIT Cardinal Red (`#A31F34`) accent color
- 880px max content width

### Architecture Patterns
- Static site generated by Jekyll
- Content separated from layout (Markdown + Liquid templates)
- Data-driven sections via `_data/` YAML files (team, talks, publications)
- Collection-based content for news posts (`_posts/`)
- Responsive design with mobile breakpoint at 640px

### Testing Strategy
- Local preview via `bundle exec jekyll serve`
- Visual verification in browser
- All pages should return HTTP 200
- Link checking before deploy

### Git Workflow
- `main` branch auto-deploys to GitHub Pages
- Feature branches for content and layout changes
- PRs for structural changes

## Domain Context
- **Archi**: Open-source RAG framework (https://github.com/archi-physics/archi)
- **CMS**: Compact Muon Solenoid experiment at CERN's Large Hadron Collider
- **Affiliated institutions**: MIT, CERN, Fermilab, University of Chicago, University of Wisconsin-Madison
- **Audience**: Physicists, computing operations engineers, academic researchers, potential collaborators
- **Previous website**: https://ppc.mit.edu/a2/ (WordPress, outdated)

## Important Constraints
- Must be hostable on GitHub Pages (no server-side processing)
- Professional tone befitting academic/research institutions (MIT, CERN, Fermilab)
- Minimal and clean design — no visual clutter
- Accessible and responsive (mobile-friendly)
- MIT License (consistent with the Archi project)

## External Dependencies
- GitHub Pages (hosting and build)
- Jekyll (static site generator)
- Archi GitHub repository for linking to source code and docs
